# DailyMenu App
A rails app (one day a gem) designed to produce printable PDF menus and webpages with a single blow.

## Site Outline (v.1)
 login
  * current menu
  * new menu (choice of: copy prev day or blank tpl)
    - * print/export: PDF
    - * update website
    - * fb post
    - * tw post
  * print/export: PDF
    - * menu archive
  * menu archive

## Languages
 HTML5, CSS3, Rails, Ruby, Slim, Sass
